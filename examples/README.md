# Examples

This directory contains example scripts demonstrating how to use the homunc library.

## Available Examples

### basic_example.py

A comprehensive example showing:
- Basic homeostatic system creation and simulation
- Adding uncertainty to systems
- Stability analysis
- Visualization of results

To run:
```bash
cd examples
python basic_example.py
```

## Requirements

The examples require:
- numpy
- matplotlib (for visualizations)

Install with:
```bash
pip install numpy matplotlib
```
